**Wirekrak Lite** is designed for application developers and data consumers who want access to domain-level market data without dealing with protocol mechanics or infrastructure concerns.

Lite provides a minimal, conservative, exchange-agnostic API with safe defaults and explicitly limited guarantees. It deliberately restricts access to low-level concepts in order to reduce cognitive load and misuse. The scope, guarantees, and limitations of Lite are formally defined in the Lite Audience Contract.
