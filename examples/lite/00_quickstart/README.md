## Available Quickstart Examples

These examples provide the fastest way to verify that Wirekrak Lite is working
and that you can receive market data.

- `trades.cpp`  
  Minimal example subscribing to trade updates.

- `book.cpp`  
  Minimal example subscribing to order book updates.

Both examples demonstrate the same client lifecycle and can be used interchangeably
depending on the type of market data you are interested in.
